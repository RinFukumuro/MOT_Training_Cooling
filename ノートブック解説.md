## 1. `ビームパラメータGaussianフィッティング-2.ipynb`

### 全体の目的

このノートブックは、MOT 用レーザーのビームをレンズで整形するときに、

- もともとのビームがどんなガウシアンビームか
- レンズをどこに置くと、どこに新しいウェストができるか
- x 方向と y 方向の両方で、実験に使いやすい長さがどれだけ確保できるか
- 手持ちのレンズの中で、どの組み合わせが現実的か

を順番に調べるためのノートブックです。

- `numpy`: 数値計算

- `curve_fit`: 測定データをガウシアンビーム式にフィット

つまり、このセルは「解析に必要な道具の準備」です。

物理的な意味:

- x と y のウェスト位置がずれている
- 半径もかなり違う

つまり、もとのビームは円形ではなく楕円で、しかも非点収差的なずれを持っています。

このノートブックは、実測したビーム径から出発して、

- 出射直後のガウシアンビームパラメータを推定する
- 1 枚レンズを置いたときにどう変わるか

### Cell 0

基本ライブラリを読み込み、ガウシアンビーム半径の式と補助関数を定義しています。

関数の意味:

- `gaussian_beam_radius(z, w0, z0)`
  位置 `z` におけるビーム半径
- `linear(z, theta, r0)`
  発散角の線形近似
- `truncate_n(x, n)`
  小数点以下を丸める補助

主役は `gaussian_beam_radius` です。

### Cell 3

出射後のビーム半径データを x, y 方向で入力し、
ガウシアンビームにフィットしています。

やっていること:

1. 測定位置 `z` を定義
2. x, y の半径データを入力

- 実部: ウェスト位置

これを使うと、レンズや自由空間伝搬を ABCD 行列で簡単に追跡できます。

## 4. 研修向けに押さえると分かりやすいポイント

1. まず「ガウシアンビームの 3 つの基本量」を押さえる

- `w0`: 最も細い半径
- `z0`: その位置
- `zR`: そこからどれくらい平行光に近いまま保てるか

2. 次に「q パラメータ」を理解する

- `q = -z0 + i zR`
- これを使うと、レンズや自由空間伝搬を ABCD 行列で一気に計算できる

3. 最後に「x と y を別々に見る理由」を理解する

- 実ビームは理想的な円形とは限らない
- x と y でウェスト位置や大きさが違う
- その差が非点収差やカップリング効率低下につながる

この 3 段階で見ると、2 つのノートブックの流れがかなり追いやすくなります。
