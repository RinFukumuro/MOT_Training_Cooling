# Sellmeier コード解説

この資料は、以下のコードが何をしているのかを、

- 物理的な意味
- Python の構文
- 各行の役割

の 3 つの観点から詳しく説明するためのメモです。

```python
#Sellemeier Equation
def sellmeier(lam):
    return np.sqrt(1+0.6961663*lam**2/(lam**2-0.0684043**2)+0.4079426*lam**2/(lam**2-0.1162414**2)+0.8974794*lam**2/(lam**2-9.896161**2))

lam_list = np.arange(0.2,3.8,0.01) # [um]
plt.plot(lam_list,sellmeier(lam_list))
plt.xlim(0.2,3.8)
plt.ylim(1.35,1.60)
plt.xticks(np.arange(0.2,3.81,0.6))
plt.xlabel("wave length [um]")
plt.ylabel("refractive index")
plt.grid(True)
```

---

## 1. このコードは何をしているか

このコードは、`Sellmeier 式` を使って

- 波長 `lam`
- その波長に対応する屈折率 `n`

の関係を計算し、グラフとして表示しています。

つまり、

「このガラス材料は、波長が変わると屈折率がどう変わるか」

を可視化するコードです。

光学では、屈折率は一定ではなく波長に依存します。  
この性質を `分散` と呼びます。

Sellmeier 式は、この分散をよく表す経験式です。

---

## 2. 1 行ずつの説明

### 1 行目

```python
#Sellemeier Equation
```

これはコメントです。

- `#` から始まる行は、Python では実行されません
- 人が読むためのメモです

ここでは「この下に Sellmeier 式を書く」という意味です。

### 2 行目

```python
def sellmeier(lam):
```

これは関数定義です。

- `def` は「関数を定義する」という意味
- `sellmeier` は関数名
- `(lam)` は引数

ここでは、`lam` という値を受け取って、その波長に対する屈折率を返す関数を作っています。

#### 構文の意味

- `def 関数名(引数):`
  という形で関数を作る
- 行末の `:` は、「このあとに関数の中身が始まる」ことを表す
- 次の行が字下げされているのは、「この行が関数の中身です」という意味

### 3 行目

```python
return np.sqrt(1+0.6961663*lam**2/(lam**2-0.0684043**2)+0.4079426*lam**2/(lam**2-0.1162414**2)+0.8974794*lam**2/(lam**2-9.896161**2))
```

この行が Sellmeier 式そのものです。

- `return` は「この値を関数の返り値として返す」という意味
- `np.sqrt(...)` は平方根
- 全体として、屈折率 `n` を計算しています

#### 数式としての意味

この式は次の形です。

```text
n(λ) = sqrt(1 + A1 λ^2 / (λ^2 - B1) + A2 λ^2 / (λ^2 - B2) + A3 λ^2 / (λ^2 - B3))
```

ここで、

- `A1, A2, A3`
- `B1, B2, B3`

に対応する係数がコード中に入っています。

この係数は材料ごとに決まっています。

#### 構文の意味

- `np.sqrt(x)`
  NumPy の平方根関数
- `**2`
  2 乗
- `*`
  掛け算
- `/`
  割り算
- `+`
  足し算
- `(`
  `)`
  計算のまとまりを明示する

#### `lam**2` とは

```python
lam**2
```

は

```python
lam × lam
```

と同じで、`lam` の 2 乗です。

#### なぜ `np.sqrt` を使うのか

Sellmeier 式は、最後に平方根を取って屈折率 `n` を求める形だからです。

## 3. 関数の外側の処理

### 4 行目

```python
lam_list = np.arange(0.2,3.8,0.01) # [um]
```

これは、波長の配列を作っています。

- `np.arange(start, stop, step)`
  は、一定間隔の数列を作る関数です

この場合は

- `0.2`
  から
- `3.8`
  まで
- `0.01`
  刻み

で波長を並べています。

単位はコメントにある通り `um` です。

#### できる配列のイメージ

```python
[0.2, 0.21, 0.22, 0.23, ...]
```

のような値が入ります。

#### 構文の意味

- `lam_list = ...`
  は「右辺の値を左辺の変数に代入する」という意味です
- `# [um]`
  はコメントで、単位がマイクロメートルであることを示しています

---

### 5 行目

```python
plt.plot(lam_list,sellmeier(lam_list))
```

これはグラフを描いています。

- 横軸に `lam_list`
- 縦軸に `sellmeier(lam_list)`

を取っています。

つまり、

「波長ごとの屈折率」

を曲線として表示しています。

#### 構文の意味

- `plt.plot(x, y)`
  は Matplotlib で 2 次元グラフを描く基本関数です
- `sellmeier(lam_list)`
  では、関数 `sellmeier` に配列を入れています

#### ここで大事な点

`lam_list` は配列ですが、NumPy を使っているので

```python
sellmeier(lam_list)
```

と書くと、配列の各要素に対してまとめて計算できます。

これを `ベクトル化された計算` と考えると分かりやすいです。

---

### 6 行目

```python
plt.xlim(0.2,3.8)
```

これは x 軸の表示範囲を指定しています。

- 最小値 `0.2`
- 最大値 `3.8`

つまり、横軸を波長 `0.2 um` から `3.8 um` の範囲で表示します。

### 7 行目

plt.ylim(1.35,1.60)

研修では、まず次の 3 点を押さえると分かりやすいです。

1. `sellmeier(lam)` は「波長を入れると屈折率を返す関数」
2. `lam_list` は「たくさんの波長を並べた配列」
3. `plt.plot(lam_list, sellmeier(lam_list))` は「波長と屈折率の対応をグラフ化」
