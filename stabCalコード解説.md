# stabCal コード解説

この資料は、`stabCal` 関数が何をしているのかを、

- 物理的な意味
- Python の構文
- 各行の役割

の 3 つの観点から説明するためのメモです。

対象コードは以下です。

```python
#stability condition

def stabCal(L,R,lam): 
    ML = np.array([[1,L],[0,1]]) #Lcavだけ直進
    MR = np.array([[1,0],[-2/R,1]]) #曲面鏡での反射

    M = np.matmul(MR,ML)
    M = np.matmul(ML,M)

    A=M[0][0]
    B=M[0][1]
    C=M[1][0]
    D=M[1][1]

    
    if D-A==0:
        R_cav = np.inf
    else:
        R_cav = 2*B/(D - A)
    w_cav = np.sqrt(2*B*lam/(np.pi*np.sqrt(4-(A+D)**2)))
    return R_cav, w_cav

R_cav, w_cav = stabCal(Lg,Rg,lam)
print(f"R_cav = {R_cav} [m]")
print(f"waist_cav = {w_cav*1e6:.2f} [um]")
```

---

## 1. このコードは何をしているか

このコードは、`ABCD 行列` を使って共振器の安定条件に関係する量を計算しています。

具体的には、

- 共振器の中での代表的な曲率半径 `R_cav`
- 共振器内でできるビームウェスト `w_cav`

を求めています。

ここでいう共振器は、光が

- ある距離だけ進み
- 曲面鏡で反射し
- また進む

という光学系として近似されています。

このセルは、後半のビーム整形計算そのものではなく、

「前段の光学系でどんなビームができそうか」

をざっくり見積もる補助的な計算です。

---

## 2. 背景にある物理

### ABCD 行列とは

ABCD 行列は、光線やガウシアンビームが光学系を通るときの変化を、`2×2 行列` で表す方法です。

例えば、

- 自由空間を進む
- レンズを通る
- 曲面鏡で反射する

といった操作を、それぞれ行列として書けます。

そして、複数の素子を通るときは、それらの行列を掛け合わせることで全体の変換が分かります。

---

### 自由空間伝搬の行列

距離 `L` だけ自由空間を進むときの行列は

```text
[ 1  L ]
[ 0  1 ]
```

です。

これは、

- 光線の角度は変わらない
- 位置だけが進行距離に応じて変わる

ということを表しています。

---

### 曲面鏡反射の行列

曲率半径 `R` の曲面鏡で反射するときの行列は、
このコードでは

```text
[ 1    0 ]
[ -2/R 1 ]
```

として書かれています。

これは、鏡の曲率によって光線の進む向きが変わることを表しています。

`-2/R` となっているのは、`レンズ` ではなく `反射` を扱っているためです。

---

### 安定条件とは

共振器の中で光が何往復しても発散しすぎず、ある決まったビーム形状を保てるとき、
その共振器は `安定` だといいます。

このコードでは、行列要素 `A, B, C, D` を使って、

- 共振器の安定性に関係する量
- そのときのビームウェスト

を計算しています。

---

## 3. 1 行ずつの説明

### コメント行

```python
#stability condition
```

これはコメントです。

- `#` 以降は実行されません
- 人向けのメモです

ここでは「安定条件に関する計算を書く」という意味です。

---

### 関数定義

```python
def stabCal(L,R,lam): 
```

これは関数定義です。

- `def`: 関数を定義する
- `stabCal`: 関数名
- `L, R, lam`: 入力引数

ここでの引数の意味は、

- `L`: 光が進む距離
- `R`: 曲面鏡の曲率半径
- `lam`: 波長

です。

つまりこの関数は、

「距離 `L`、曲率半径 `R`、波長 `lam` を与えると、共振器の代表的なパラメータを返す関数」

です。

---

### 自由空間伝搬行列

```python
ML = np.array([[1,L],[0,1]]) #Lcavだけ直進
```

これは、自由空間を `L` だけ直進する行列を作っています。

#### 構文の意味

- `np.array([...])`
  NumPy 配列を作る
- `[[1,L],[0,1]]`
  2 行 2 列の行列

つまり `ML` は

```text
[ 1  L ]
[ 0  1 ]
```

に対応します。

コメントの `Lcavだけ直進` は、「共振器長だけまっすぐ進む」という意味です。

---

### 曲面鏡反射行列

```python
MR = np.array([[1,0],[-2/R,1]]) #曲面鏡での反射
```

これは、曲率半径 `R` の曲面鏡で反射する行列です。

`-2/R` の部分が、鏡の曲率による方向変化を表しています。

この行列は

```text
[ 1    0 ]
[ -2/R 1 ]
```

に対応します。

---

### 行列の掛け算 1

```python
M = np.matmul(MR,ML)
```

これは、まず

- 自由空間を進んで
- 曲面鏡で反射する

という変換を 1 つの行列 `M` にまとめています。

`np.matmul(A, B)` は、行列 `A × B` の掛け算です。

---

### 行列の掛け算 2

```python
M = np.matmul(ML,M)
```

これは、さらにもう一度自由空間伝搬を加えています。

結果として、全体の行列 `M` は

- 進む
- 反射する
- 進む

という光路を表しています。

---

### 行列成分の取り出し

```python
A=M[0][0]
B=M[0][1]
C=M[1][0]
D=M[1][1]
```

これは、2×2 行列 `M` の各成分を取り出しています。

つまり

```text
M = [ A  B ]
    [ C  D ]
```

という形に分解しています。

#### 構文の意味

- `M[0][0]`
  1 行 1 列の成分
- `M[0][1]`
  1 行 2 列の成分
- `M[1][0]`
  2 行 1 列の成分
- `M[1][1]`
  2 行 2 列の成分

Python は 0 始まりなので、

- 0 行目 = 1 行目
- 1 行目 = 2 行目

として数えます。

---

### 条件分岐

```python
if D-A==0:
    R_cav = np.inf
else:
    R_cav = 2*B/(D - A)
```

ここでは、共振器内の代表的な曲率半径 `R_cav` を計算しています。

ただし、

```python
D - A == 0
```

だと分母が 0 になってしまうため、そのときは

```python
R_cav = np.inf
```

つまり「無限大」としています。

#### 構文の意味

- `if 条件:`
  条件が真なら以下を実行
- `else:`
  そうでなければ以下を実行
- `np.inf`
  NumPy における無限大

---

### ビームウェストの計算

```python
w_cav = np.sqrt(2*B*lam/(np.pi*np.sqrt(4-(A+D)**2)))
```

ここでは、共振器内で安定に存在できるガウシアンビームのウェスト半径 `w_cav` を計算しています。

この式は、ABCD 行列の成分から導かれる標準的な関係式です。

ポイントは、

- `B`
- `A + D`
- `lam`

からウェストが決まることです。

#### 物理的な意味

- 波長が長いほど、一般にウェストは大きくなりやすい
- 共振器の幾何学的条件によって、許されるビームサイズが決まる

---

### 返り値

```python
return R_cav, w_cav
```

この関数は 2 つの値を返しています。

- `R_cav`
- `w_cav`

Python では、このようにカンマで並べると複数の値を返せます。

## 6. このコードの出力は何を意味するか

### `R_cav`

これは、共振器内での代表的な波面曲率に関係する量です。

値が無限大に近い場合は、波面がかなり平坦に近いことを意味します。

### `w_cav`

これは、共振器内で形成されるガウシアンビームのウェスト半径です。

つまり、

「この共振器条件なら、中心でどれくらい細いビームができるか」

の目安です。
